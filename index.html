Smart India Hackathon 2024

AI-Powered Educational Management System

Our Solution

We built an AI-powered Educational Management Web App that makes scheduling and coordination effortless, smart, and dynamic!

Key Features
  • Role-based secure login using JWT Authentication (Admin | Teacher | Student)
  • AI-driven timetable generation — auto-handles class clashes, cancellations & capacity limits
  • Real-time lecture confirmations by teachers and elective selection by students
  • Instant notifications for any schedule updates
  • Responsive dashboard UI built with Bootstrap for a smooth, intuitive experience
Leading the Project

Leading this project was an amazing learning experience from ideation to implementation. Our system ensures seamless scheduling, transparent communication, and adaptive learning workflows, empowering institutions to operate smarter and faster.
